# git 명령어



| 명령                          | 내용                            | 비고 |
| ----------------------------- | ------------------------------- | ---- |
| `git init`                    | 현재 폴더에 Git Repository 생성 |      |
| `git status`                  | git 현재 상태 확인              |      |
| `git config -l`               | config 설정 확인하기            |      |
| `git add`                     | 파일 또는 폴더를 staging        |      |
| `git commit -m '커밋 메시지'` | 커밋 메시지와 함께 저장         |      |
| `git log`                     | git log 확인                    |      |
|                               |                                 |      |
