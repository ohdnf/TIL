| 명령어   | 내용                                                    |
| -------- | ------------------------------------------------------- |
| `pwd`    | 현재 작업 디렉토리 표시                                 |
| `ls`     | 현재 작업 디렉토리에 있는 디렉토리 및 파일 표시         |
| `cd`     | 디렉토리 이동(`.` : 현재 디렉토리 / `..` 상위 디렉토리) |
| `mkdir`  | 디렉토리 생성                                           |
| `code`   | `code .` : 현재 디렉토리에서 vscode 실행                |
| `python` | Python으로 파일 실행( ex. `python hello.py` )           |
| `pip`    | Python Package Installer                                |
| `rm -r`  | 디렉토리 삭제 (폴더 내 파일 Recursive하게 삭제)         |
|          |                                                         |
