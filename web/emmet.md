# Emmet
Code Editor의 자동완성 기능

## HTML emmet 사용 예시
```html
<!-- div#exampleId>div.eampleClass$*3>a.directLink+img.feature -->
<!-- 위 코드 + Tab 입력하면 아래 코드로 변환된다. -->
<div id="exampleId">
    <div class="eampleClass1"><a href="" class="directLink"></a><img src="" alt="" class="feature"></div>
    <div class="eampleClass2"><a href="" class="directLink"></a><img src="" alt="" class="feature"></div>
    <div class="eampleClass3"><a href="" class="directLink"></a><img src="" alt="" class="feature"></div>
</div>
```
