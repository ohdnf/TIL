# JavaScript 문법

## 1. 연산자
- 산술 연산자: 덧셈, 뺄셈, 곱셈, 나눗셈, 나머지(`%`) 연산
- 할당 연산자: `+=`, `-=`, `*=`, `/=`, `%=`
- 동등 연산자: `==`
    ```js
    0 == ''     // true
    0 == '0'    // true
    '' == '0'   // false
    ```
- 일치 연산자: `===`
    ```js
    0 === ''
    0 === '0'
    '' === '0'
    ```
- 부등, 불일치 연산자: `!=`, `!==`
- 비교 연산자: `<`, `>`, `<=`, `>=`
- 논리 연산자: `&&`, `||`, `!`
- 기타 연산자

## 2. 조건문과 반복문

### 1) 조건문
- `if`문: `if`, `else if`, `else`
- `switch`문
    ```js
    var s;
    switch(s){
        case:
        break;
    }
    ```

### 2) 반복문
- `for`문
    ```js
    for (i=0; i<10; i++) {
        // do something
    }
    ```
